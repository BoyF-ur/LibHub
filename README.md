## 🔮 Định hướng phát triển

### 🚀 1. Nâng cấp UX/UI
- Gợi ý tìm kiếm thông minh  
- Bộ lọc nâng cao  
- Fix UI/UX (Header, layout, image)

### 📚 2. Quản lý mượn sách
- ⏰ Deadline + nhắc trả sách  
- 💰 Tính phí trễ hạn  
- 📅 Chọn ngày mượn/trả hợp lý  

### 💬 3. Confession nâng cao
- React bài viết 👍👎  
- Tag sách  
- Top review nổi bật  

### ⚙️ 4. Admin System
- Quản lý theo người mượn  
- Theo dõi lịch sử  
- Request thêm sách  

### 🤖 5. Smart Library
- Gợi ý sách theo sở thích  
- Phân loại theo ngành/kỳ học  
- Thông báo khi có sách
